Manuel d'utilisation
Introduction

Notre module Tasks permet une organisation et suivie des tâches au sein d'un groupe ou d'une société. Un utilisateur peut creer une tâche et cloturer une tâche.

Le module dispose sur sa page d'accueil une representation des activités d'un utilisateur donné. L'administrateur peut avoir une vue sur la performance de chaque utilisateur.

Le module envoie automatiquement un message au responsable de la tâche (lorsqu'une tâche est crée), et à l'auteur de la tâche (lorsqu'une tâche est clôturée).

Créer une tâche
Pour créer une tâche, suivez l'ordre suivant:
  • Cliquez sur le lien Tâches crées dans le menu situé en haut de la page.
  • Sur la page, vous avez à droite un tableau listant toutes les tâches crées par vous. Et à gauche le formulaire pour l'enrégistrement d'une tâche.
  • Remplissez le forlumaire et Enrégistrer
Afficher une tâche

Pour afficher les détails d'une tâche, cliquez sur l'icone qui se trouve sur la même ligne que la tâche en question. Vous acceder a une page qui vous présente toutes les informations de la tâches.

Modifier une tâche

Pour modifier les information d'une tâche, cliquez sur l'icone qui se trouve sur la même ligne que la tâche en question. Dans le formulaire situé à gauche sur la page, vous pouvez modifier les informations et Enrégistrer.

Clôturer une tâche
Pour clôturer une tâche, Veuillez suivre l'ordre suivant:
  • Cliquez sur le lien Tâches reçues.
  • cliquez sur l'icone qui se trouve sur la même ligne que la tâche en question.
  • Dans le formulaire situé à gauche sur la page, vous pouvez remplir les informations et Enrégistrer.
Gérer votre compte
Pour gerer votre compte, veuillez suivre l'ordre suivant:
  • Cliquez sur Configuration dans le menu.
  • Sélectionner Mon compte pour acceder à votre page.
  • Vous pouvez consulter ou modifier vos informations sur la page.
Ajouter des utilisateurs
Pour ajouter un utilisateur,Rassurez-vous que vous êtes Administrateur et veuillez suivre l'ordre suivant:
  • Cliquez sur Configuration dans le menu.
  • Sélectionner Réglage pour acceder à la page.
  • Vous pouvez remplir les informations de l'utilisateur et enrégistrer.
